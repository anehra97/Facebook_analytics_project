# Facebook Post Analytics — SQL Project

## Overview

This project contains a collection of SQL queries designed to analyze social media performance using a simulated **Facebook Post Analytics** dataset.

The objective of this project is to demonstrate practical SQL skills used in real-world analytics workflows, including filtering, aggregation, grouping, and performance analysis across multiple business dimensions.

---

## Dataset Description

The dataset `facebook_post_analytics` represents social media performance metrics for Facebook posts.

### Key Fields

* `country` — Country where engagement originated
* `language` — Post language
* `user_name` — Content creator
* `content_category` — Category of content (Gaming, Finance, Fitness, etc.)
* `post_type` — Video, Reel, etc.
* `post_date` — Date of post publication
* `reach` — Number of users reached
* `likes` — Total likes
* `shares` — Total shares
* `comments` — Total comments
* `engagement_time_minutes` — User engagement duration

---

## Project Goals

This project focuses on answering common analytics questions such as:

* Which countries generate the highest reach?
* What content categories drive engagement?
* Which creators perform best?
* How engagement varies across languages and regions
* Performance comparison across post types

---

## SQL Concepts Demonstrated

* Filtering using `WHERE`
* Pattern matching with `LIKE`
* Date range analysis
* Aggregations (`SUM`, `AVG`)
* Grouped analysis using `GROUP BY`
* Post-aggregation filtering with `HAVING`
* Multi-dimensional reporting
* Query formatting best practices
